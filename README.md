#  AI Health Co-Pilot
An AI-powered food ingredient analyzer that helps users understand what's in their food at the moment of decision.
    Problem Statement
When shopping for food, consumers face ingredient lists filled with unfamiliar names and scientific terms. At the exact moment they need clarity, they're left guessing about what these ingredients mean for their health and dietary choices.
   Solution
Our AI Co-Pilot reimagines the ingredient analysis experience by:

No configuration needed: Simply paste the ingredient list
Instant insights: AI infers what matters to you
Plain language: Explains trade-offs and uncertainties clearly
Confidence, not confusion: Reduces cognitive load at decision time

   Features

- Paste any ingredient list for instant analysis
- AI-powered contextual understanding
- Explains why ingredients matter (not just lists them)
- Highlights trade-offs and scientific uncertainty
- Clean, distraction-free interface
- No login required
- Privacy-focused (no data storage)

   Technology Stack

Frontend: React 18 with Hooks
Styling: Tailwind CSS
AI: Claude API (Anthropic)
Icons: Lucide React
Deployment: Static hosting (Vercel, Netlify, GitHub Pages)

   Prerequisites
Before you begin, ensure you have:

Node.js (v16 or higher) installed
npm or yarn package manager
A Claude API key from Anthropic (https://console.anthropic.com/)
